# Modifier le rôle d'un utilisateur

Ce guide explique comment modifier le rôle d'un utilisateur dans votre espace de travail.

Note:* Seuls les utilisateurs ayant le rôle Admin ou Lead peuvent changer le rôle d'un autre utilisateur.

  1. Accédez à votre espace de travail
  2. Cliquez sur le lien "Voir les membres" dans le coin supérieur droit de la page.
  3. Cliquez sur la liste déroulante du rôle de l'utilisateur dont vous souhaitez modifier le rôle.
  4. Sélectionnez le nouveau rôle dans la liste déroulante
  5. Cliquez sur "Enregistrer les modifications" en haut de la page.
