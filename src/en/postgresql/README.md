# PostgreSQL on the Federal Science DataHub

PostgreSQL is a fully managed relational database service based on the popular open-source PostgreSQL database engine. It provides high availability, scalability, and security for your data. Azure PostgreSQL is ideal for:

* storing structured data
* running transactional workloads
* building applications that require relational databases
* managing data for web applications 

The Federal Science DataHub (FSDH) allows you to provision Azure PostgreSQL for you research, enabling you to store and manage your data in a secure and scalable environment.

Learn how to:

* Provision PostgreSQL on the FSDH: [Requesting, configuring and removing tools in your workspace](/en/managing-workspaces-and-users/Request-tools)
* Add users to PostgreSQL: [Add a FSDH user to PostgreSQL](/en/postgresql/Postgres-Add-User)
* Configure the allowlist for PostgreSQL: [PostgreSQL Configuration](/en/postgresql/Postgres)
* Compare PostgreSQL with Databricks database features: [PostgreSQL vs Azure Databricks - Database Features](/en/postgresql/psql-databricks-comparison)
