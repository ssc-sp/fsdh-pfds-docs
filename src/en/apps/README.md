# Web Applications on the Federal Science DataHub

Azure App Service is a fully managed platform for building, deploying, and scaling web apps. It provides a rich set of features for developing web applications in a variety of programming languages using Docker containers. Azure App Service on the Federal Science DataHub (FSDH) for:

* hosting web applications
* building APIs
* deploying containerized applications
* scaling web applications

The FSDH allows you to provision Azure App Service for your applications, which will be accessible to other researchers within the workspace.

Learn how to:
* Provision App Service on the FSDH: [Requesting, configuring and removing tools in your workspace](/en/managing-workspaces-and-users/Request-tools)
* Host your own web applications on the FSDH: [Host Web Apps on FSDH](/en/webapps/WebApps)
